Help

This is a simple description of how to use the PDF Word Highlighter + Piper TTS app:

  • Open a PDF using the folder icon:
  • Click sentences to play them using TTS.
  • Adjust voice and speed from the dropdowns.
  • Press Space to toggle Play/Pause
  • Highlight text using the highlighter button or press H.
  • Use Arrows to go to next/previous phrase
